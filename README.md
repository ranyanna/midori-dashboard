# Orbit

Dashboard de controle financeiro pessoal: registre entradas e saídas, acompanhe seu saldo e visualize seus gastos por categoria em gráficos.

🔗 [Ver projeto ao vivo](https://ranyanna.github.io/orbit-dashboard/)

## Funcionalidades

- Cadastro de transações (entrada ou saída) com categoria e data
- Saldo, total de entradas, saídas e percentual de economia
- Gráficos de evolução mensal e de gastos por categoria
- Dados salvos no navegador (`localStorage`)
- Layout responsivo para mobile

## Tecnologias

HTML5 · CSS3 · JavaScript · [Chart.js](https://www.chartjs.org/)

## Como rodar

Abra o projeto com a extensão **Live Server** do VS Code (não abra o `index.html` direto pelo navegador).

```
git clone https://github.com/ranyanna/orbit-dashboard.git
```
